hiit_timer
==========

Web timer for HIIT training program

usage
=====

example: 
sequence: ```20 plank jacks, +10, 20 plank leg raises, +60```
reps: ```3```

translates to:

3 repetitions of:

* 20 seconds plank jacks
* 10 seconds rest
* 20 seconds plank leg raises
* 60 seconds rest
